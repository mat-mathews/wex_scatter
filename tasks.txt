PHASE 1: GRAPH MODEL + BULK BUILDER (HIGH PRIORITY)
- [ ] Create scatter/core/graph.py — ProjectNode, DependencyEdge, DependencyGraph
      DependencyGraph is a pure data structure (mutation, query, traversal, serialization).
      All analysis algorithms are standalone free functions (SRP — no god object).
      Per-node edge indexes (_outgoing/_incoming) for O(degree) lookups.
      Evidence lists capped at 10 entries with evidence_total count.
      ProjectNode stores file_count (not full file paths — not serialized to cache).
- [ ] Create scatter/analyzers/graph_builder.py — build_dependency_graph()
      Single-pass O(P+F) construction: parse all .csproj once, scan all .cs once.
      Reverse index for .cs-to-.csproj mapping (sorted prefix match, not per-file dir walk).
      Reuses: find_files_with_pattern_parallel, derive_namespace, extract_type_names_from_content.
- [ ] Modify scatter/scanners/project_scanner.py — add parse_csproj_all_references()
      Extracts ALL ProjectReferences, namespace, framework, output type, project style.
- [ ] Create test_graph.py — ~39 tests (nodes, edges, graph ops, builder, csproj parser, dir index)

PHASE 2: METRICS + CYCLE DETECTION (HIGH PRIORITY)
- [ ] Create scatter/analyzers/coupling_analyzer.py — all standalone free functions
      ProjectMetrics: fan_in, fan_out, instability, coupling_score, shared_db_density, etc.
      compute_all_metrics(graph) — O(N+E) using per-node edge indexes.
      Configurable coupling_weights (default: project_ref=1.0, sproc=0.8, namespace=0.5, type=0.3).
      Weights documented with rationale, overridable in .scatter.yaml.
      rank_by_coupling(metrics, top_n) for top-N most coupled projects.
- [ ] Cycle detection via Tarjan's SCC algorithm — O(N+E)
      CycleGroup dataclass: projects, size, shortest_cycle, edge_count.
      detect_cycles(graph) returns List[CycleGroup] (SCCs with size > 1).
      For each SCC, extract one shortest cycle via BFS for illustration.
      NOT DFS back-edge (finds only *a* cycle), NOT full enumeration (exponential).
- [ ] Create test_coupling.py — ~20 tests (metrics, custom weights, SCC, shortest cycles)
